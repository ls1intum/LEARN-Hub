# Getting Started with LEARN-Hub

## Accessing LEARN-Hub

Visit LEARN-Hub in your web browser. No installation required – it works on any device with internet access.

## Guest vs Authenticated Access

| Feature | Guest | Teacher | Admin |
|---------|-------|---------|-------|
| Browse activities | ✓ | ✓ | ✓ |
| Create recommendations | ✓ | ✓ | ✓ |
| Save favorites | — | ✓ | ✓ |
| View history | — | ✓ | ✓ |
| Manage users | — | — | ✓ |
| Upload activities | — | — | ✓ |

Use **Guest mode** to explore without creating an account. **Teachers** can create an account to save favorites and maintain search history.

## Creating an Account

1. Click **Sign Up** on the login page
2. Enter your email address and name
3. The system generates a secure password and sends it to your email
4. Check your email for your credentials
5. Log in with your email and the password provided

## Logging In

### Option 1: Password Login
Enter your email and password.

### Option 2: Verification Code Login
1. Enter your email address
2. Check your email for a 6-digit verification code
3. Enter the code in the app (valid for 10 minutes)
