# Create Activity Recommendations

import useBaseUrl from '@docusaurus/useBaseUrl';

The **Recommendation Configurator** is the primary entry point of the LEARNApp. It allows teachers to generate activity sets based on classroom context and teaching goals.

## Overview

- Guided, multi-step process to personalize recommendations
- Parameters include: Duration, Grade, Topic, and Available Devices
- Results display the most relevant and age-appropriate activities

## Steps for Creating Activity Recommendations

1. Open the **LEARNApp Planung** tab from the navigation bar.
